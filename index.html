Secure Wallet – Privacy Policy

Overview

Secure Wallet is a Telegram-based application designed with a strong focus on user privacy and data security. We collect only the minimum information required to operate the service.

Information We Collect
  • Telegram User ID and username
  • Wallet balance and transaction records
How We Use Data
  • To provide wallet and payment functionality
  • To prevent abuse and ensure system security
What We Do Not Do
  • We do not sell user data
  • We do not collect personal documents
  • We do not share data with third parties

Data Security

All stored data is protected using standard security practices. Access is limited and data is used strictly for operational purposes.

Policy Updates

This Privacy Policy may be updated periodically. Continued use of Secure Wallet indicates acceptance of the updated policy.
